Qu'est-ce que la table ASCII ? -> ASCII : permet de transformer les lettres et symboles en nombres. Un certain ordre a été attribué à la table ASCII "ASCII ALPHABÉTIQUE”. Code ASCII = 2^7 symboles soit 128

La table ASCII est générale mais que faire si un alphabet est plus grand qu’un autre? Une nouvelle langue ISO a donc été inventée dont les variantes divergent selon les besoins de chaque langue. Ainsi la nouvelle variante générale de ASCII est l’UTF-8. Aujourd’hui avec UTF-8 l’on peut presque tout coder sur le web.
